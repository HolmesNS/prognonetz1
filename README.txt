1. Install Flask.

2. Run Flask Sever using following commands:

For windows cmd run following command:

    set FLASK_APP=server.py
    flask run --host=0.0.0.0

For windows power shell

    $env:FLASK_APP = "server.py"
    flask run --host=0.0.0.0

And Visit http://127.0.0.1:5000/ to view the web application.
